## Hi, I'm Pedro Paulo Macedo Neto 👋

Tech Lead and M.Sc. in Computer Science with 8+ years of experience building multiplayer, XR, and real-time 3D applications with Unity and Unreal.

I currently lead the development of a multiplayer educational platform, where I:
- architect and implement the full networking layer using **NGO (Netcode for GameObjects)** and **RPCs**
- manage **CI/CD pipelines** for **Windows, macOS, and Oculus Quest 2/3**
- operate a **headless server on AWS EC2** for client communication and user allocation
- support the team through sprint leadership, technical decisions, internal tooling, and infrastructure
- built a modular **3D asset bundle library** integrated with backend services to deliver class-specific content

I’m passionate about game development, simulators, serious games, XR, and scalable real-time systems.
